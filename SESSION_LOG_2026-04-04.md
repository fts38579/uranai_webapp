## このセッションで実施したこと

### 1. 非スピリチュアル方針への切り替え

- `uranai-v5.html` のトップコピーから `霊感` `Spiritual Guide` などの表現を削除
- AIプロンプトからスピリチュアル・運命論の方向へ寄る指示を除去
- ルノルマンを `課題・見落とし・注意点 7割 / 改善傾向 3割`
- 数秘オラクルを `前向きな助言・行動支援 9割 / 課題指摘 1割`
- 補助質問、カード定義、フォールバック文のスピリチュアル語彙も整理

### 2. 3本柱コンセプトの反映

- トップ画面コピーを 3本柱ベースへ更新
- `BRAND_PROFILE` / `MEMBERSHIP_PLAN` の文言を 3本柱ベースへ更新
- `HANDOFF_FOR_CODEX.md` にコンセプト3本柱と設計整理を追記

### 3. トップページ文言の全面リライト

- トップのサブタイトルを手法説明から価値説明へ変更
- ヒーロー文を `深く向き合う / 現実へ返す / 記録に残す` の流れで全面調整
- `guide` / `offer` / `vault` / `member` の各文言を、内部語を減らしてユーザー便益ベースへ整理
- `会員価値の土台` などの内部寄り見出しを、初見でも意味が入る表現へ更新

### 4. 追加質問フローの再設計

- 旧来の「カードごとに3問」ではなく、鑑定を着地させる `3〜5問` の可変設計へ変更
- 役割は `核心確認 / 詰まり確認 / 気持ち確認 / 外部要因確認 / 着地点確認 / 行動制約確認` の中から出題
- ルノルマンの位置情報とオラクルの流れを両方使って質問を生成
- 補足回答は `CLARIFY_ANSWERS` に役割ベースで保存し、AIプロンプトへは `補足整理` として構造化して注入
- 旧履歴の `clarifyAnswers` 形式も読み込める互換処理を追加
